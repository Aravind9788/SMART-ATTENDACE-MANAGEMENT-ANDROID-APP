# 📱 Smart Attendance Management System

A **Smart Attendance Management Android Application** developed for **Government Arts College (Autonomous), Salem**.
This project is created as my **Final Year College Project** to digitize and simplify the process of recording and managing student attendance.

## 📖 About the Project

Traditional attendance systems using paper registers are time-consuming and difficult to maintain.
This application provides a **digital solution** that allows staff to record, store, and monitor attendance efficiently using a mobile application.

The system is designed to improve **accuracy, accessibility, and management of attendance data**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Aravind9788/SMART-ATTENDACE-MANAGEMENT-ANDROID-APP.git
```

### 2️⃣ Navigate to Project

```bash
cd SMART-ATTENDACE-MANAGEMENT-ANDROID-APP
```

### 3️⃣ Install Frontend Dependencies

```bash
npm install
```

### 4️⃣ Start the Expo Application

```bash
npx expo start
```

---

## 📱 Build Android APK

To build an Android APK using Expo:

```bash
npx expo prebuild
npx expo run:android
```

Or using **Expo EAS build**:

```bash
npx expo login
npx expo build:android
```
